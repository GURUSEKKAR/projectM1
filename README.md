## CI/CD Security Pipeline Flow Chart (Node.js)

This is a small Node.js + Express application that displays your CI/CD security pipeline flow chart in the browser.

### Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the server:

   ```bash
   npm start
   ```

3. Open the app in your browser:

   ```bash
   http://localhost:3000
   ```

You can customize the flow chart by editing the HTML and styles in the `public/index.html` and `public/styles.css` files.
